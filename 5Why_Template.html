Am I Treating a Symptom or the Root Cause?
❌ Treating Symptoms
• "We need to retrain the drivers"
• "We'll add more inspections"
• "Let's send a reminder email"
• "We fired the person responsible"

These feel productive but the problem comes back because the system that created it is unchanged.
→
✓ Fixing Root Causes
• "The SOP doesn't cover this scenario — update it"
• "The checklist is missing a step — add it"
• "There's no feedback loop — build one"
• "The process allows this error — redesign it"

These change the system so the problem can't happen again, regardless of who's doing the work.
